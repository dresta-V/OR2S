# OR2S
A comprehensive IRCTC website with enhanced accessibility via voice search 🎙️ and multilingual support 🌐. Uses DFS and BFS for optimal route planning 🛤️. Integrates services like parcel booking 📦, medical aid 🚑, food-on-track 🍱, and coolie booking 🧳. Advanced algorithms ensure seamless seat allotment &amp; onboard emergency services 🚆
